# 📚 Muhaimin Pustakālay (N3X Blogger Template)

Sebuah template **Blogger (XML)** modern bernuansa _Glassmorphism_ yang dirancang khusus untuk membuat katalog perpustakaan digital, koleksi buku, atau arsip literatur pribadi. Proyek ini mengutamakan antarmuka yang bersih (UI), pengalaman pengguna yang mulus (UX), dan responsivitas di berbagai perangkat.

---

## ✨ Fitur Utama

- **🎨 Desain Glassmorphism:** Elemen antarmuka transparan menyerupai kaca dengan efek _blur_ yang elegan.
- **🌗 Mode Gelap & Terang (Dark/Light Mode):** Terintegrasi dengan penyimpanan lokal (_localStorage_) agar preferensi pengunjung tetap tersimpan.
- **🌐 Dukungan Multi-Bahasa (ID & HI):** Fitur pergantian bahasa instan (Indonesia & Hindi) di sisi klien tanpa perlu memuat ulang halaman.
- **⚡ AJAX Pagination:** Navigasi perpindahan halaman koleksi buku (Next/Prev/Angka) yang mulus tanpa _reload_ (berasa seperti Single Page Application).
- **🎠 Hero Carousel Sinematik:** _Slider_ otomatis di Beranda menggunakan [Splide.js](https://splidejs.com/) yang menonjolkan buku dengan efek _blur_ dan _scale_.
- **📱 Responsif Penuh:** Tampilan menyesuaikan secara otomatis dari layar desktop ultra-lebar hingga ponsel pintar.
- **🔍 Empty State Elegan:** Tampilan peringatan cantik (beserta tombol aksi) apabila buku yang dicari tidak ditemukan.

## 🚀 Cara Pemasangan (Instalasi)

1. Unduh atau salin seluruh isi dari file `index.xml`.
2. Buka **Dasbor Blogger** Anda.
3. Masuk ke menu **Tema (Theme)**.
4. Klik ikon panah bawah di sebelah tombol Sesuaikan (Customize), lalu pilih **Edit HTML**.
5. Hapus semua kode bawaan yang ada, lalu _paste_ (tempel) kode `index.xml` dari proyek ini.
6. Klik **Simpan**.
7. Buka menu **Setelan (Settings)** > **Postingan (Posts)** dan ubah "Jumlah maksimal postingan" menjadi **14** (atau jumlah yang Anda atur di skrip).
8. Cara input buku

## 🛠️ Teknologi yang Digunakan

- **Blogger XML Tag:** Engine utama rendering data.
- **HTML5 & CSS3:** Struktur dan gaya antarmuka (_CSS Variables, Flexbox, CSS Grid_).
- **Vanilla JavaScript:** Logika fungsionalitas (_Dark mode, Language toggle, AJAX Fetch_).
- **Splide.js:** Pustaka ringan untuk _slider carousel_.
- **Google Fonts & Material Icons:** Tipografi dan ikon _outlined_.
